# Copiloto de Vendas com IA para Cursos de Tecnologia

## Objetivo

Este projeto demonstra a criação de um Copiloto de Vendas com Inteligência Artificial para auxiliar consultores comerciais na venda de cursos de tecnologia.

O sistema ajuda o vendedor a:

* Identificar necessidades do cliente;
* Recomendar cursos adequados;
* Responder dúvidas frequentes;
* Tratar objeções;
* Sugerir próximos passos no atendimento.

## Problema Resolvido

Muitos vendedores iniciantes possuem dificuldade para:

* Entender o perfil do cliente;
* Escolher o curso ideal;
* Responder objeções;
* Conduzir o fechamento da venda.

O copiloto atua como assistente inteligente durante o atendimento.

## Abordagem Utilizada

Foi utilizada a abordagem de Copiloto de Vendas baseada em Engenharia de Prompt e Base de Conhecimento.

A IA consulta informações dos cursos, perguntas frequentes e objeções para gerar respostas consistentes.

## Base de Conhecimento

* produtos.md
* perguntas-frequentes.md
* objecoes.md

## Exemplo de Atendimento

Cliente:

"Quero aprender programação mas nunca programei."

Resposta sugerida:

"Ótimo! Para quem está começando recomendamos o Curso Java Fundamentos. Ele foi desenvolvido para iniciantes e ensina desde lógica de programação até orientação a objetos."
